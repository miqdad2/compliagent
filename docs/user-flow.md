# User Flow

1. User signs in.
2. User creates a project with client, discipline, and review type.
3. User uploads one or more documents and assigns each document a role.
4. Document processing extracts page-preserving chunks.
5. AI agents extract requirements and evidence.
6. Compliance comparison generates draft findings.
7. Reviewer validation checks evidence support.
8. Human reviewer edits, approves, or requests more information.
9. Reports and chat summaries are generated from source-backed findings.
